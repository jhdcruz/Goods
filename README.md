# Memo App

A simple to-do app.

## Tech Stack

- Kotlin
- Jetpack Compose
- Firebase
    - Firestore
    - Authentication
    - Storage
    - Crashlytics
    - Analytics
- Coil
- Dagger Hilt

## Development

Set up firebase environment in `local.properties`.

```properties
# Backend for authentication
gcp.web.client=
gcp.web.secret=

# Firebase Client SDK
gcp.client.debug=
gcp.client.release=
```
